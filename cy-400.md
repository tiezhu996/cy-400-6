## 项目主要需求

### 项目概述
**项目类型：桌面应用（含GUI）**
搭建一个读书笔记管理桌面应用，用户可在本地管理电子书笔记、高亮摘录、标签分类，支持 Markdown 编辑和全文搜索。

### 功能需求

1. **书籍管理**：添加书籍信息（书名、作者、出版社、ISBN、封面图、分类标签），支持按书名/作者/标签搜索和筛选。
2. **笔记编辑器**：内嵌 Markdown 编辑器，支持富文本编辑（加粗/斜体/列表/代码块/图片插入），实时预览。
3. **高亮摘录**：记录书中的精彩段落摘录，关联书籍和页码，支持为摘录添加个人批注。
4. **标签体系**：为书籍和笔记添加多级标签（如"文学/小说/科幻"、"技术/前端/React"），支持按标签树状浏览。
5. **全文搜索**：跨书籍、笔记、摘录的全文搜索，搜索结果高亮关键词并显示上下文。
6. **阅读统计**：阅读进度追踪（已读/在读/想读），月度阅读数量统计图表，阅读时长记录。
7. **数据导入导出**：支持从 CSV/JSON 导入书籍和笔记数据，导出笔记为 Markdown 文件或 JSON 备份。
8. **深色/浅色主题**：支持主题切换，记住用户偏好。

### 技术要求

- 框架：Electron + React 18 + TypeScript
- UI 库：Tailwind CSS + Radix UI
- Markdown 编辑器：@uiw/react-md-editor
- 全文搜索：FlexSearch（前端内存搜索）
- 本地数据库：SQLite（通过 better-sqlite3）
- 状态管理：Zustand
- 构建工具：Vite + electron-vite
- 代码规范：严格遵循单一职责原则，每个文件只负责一个功能；常量（错误码、标签分类、阅读状态枚举）统一维护在 `src/constants/` 目录下；数据库操作统一封装在 `src/database/`；所有工具函数封装在 `src/utils/`。

## 代码存放规则

Electron 桌面应用项目。

```
desktop/
├── src/
│   ├── main/               # Electron 主进程
│   │   ├── index.ts        # 主进程入口
│   │   ├── ipc/            # IPC 通信处理
│   │   └── database/       # SQLite 数据库操作
│   ├── renderer/           # 渲染进程（React 应用）
│   │   ├── pages/          # 页面视图
│   │   ├── components/     # 通用组件（编辑器、搜索栏、标签树等）
│   │   ├── hooks/          # 自定义 Hooks
│   │   ├── stores/         # Zustand 状态管理
│   │   ├── constants/      # 常量定义
│   │   ├── types/          # TypeScript 类型
│   │   ├── utils/          # 工具函数
│   │   └── styles/         # 全局样式
│   └── preload/            # preload 脚本
├── resources/              # 应用图标等资源
├── package.json
└── electron.vite.config.ts
```
